# C++ Basics Tutorial

Welcome to your C++ learning journey! This directory contains a structured path to mastering the basics of C++. Each folder corresponds to a key concept and contains executable code examples.

## How to Use This Tutorial
1.  Navigate to a concept folder (e.g., `01_Introduction`).
2.  Open the `.cpp` files to read the code and comments.
3.  Compile and run the code to see it in action.

## Quick Start
To run a C++ program from your terminal:
```bash
g++ filename.cpp -o output_name
./output_name
```
